Quick Fixes

If Flask is not running (Port 3050 fails):

  1. Open Command Prompt as Administrator
  2. Navigate to: cd C:\inetpub\wwwroot\scraper
  3. Run: start-flask-service.bat

If IIS Proxy fails (502 Bad Gateway):

  1. Open PowerShell as Administrator
  2. Navigate to: cd C:\inetpub\wwwroot\scraper
  3. Run: .\fix-iis-proxy.ps1
